对工具理性的反思。
AI与人的工具化。
1. AI的本质是人类工具理性（经验主义）的延伸。
2. AI在解放人类的智力的同时，给人类造成了巨大冲击，许多社会分工可能被AI承担。
3. AI是现代文明的必然产物，但它却冲击着现代性的底层哲学，这透露出现代性自身的内在矛盾。
4. AI给工具化的人类带来的危机感，无疑昭示着现代性的底层危机：人若不是工具，人是什么？现代哲学无法给出令人满意的答案。
5. AI具有对现代的哲学、社会的巨大（潜在）冲击，我们不得不思考几个问题：
   1. 若非工具，人是什么？
   2. 教育会变成什么样子？
   3. ...？
6. AI既解放人类智力的同时，会造成更大的财富、权力极化。
7. 如何面对AI？
   1. 接受。
   2. 使用。
   3. 警惕。
   4. 反思。
